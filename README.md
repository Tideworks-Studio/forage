## Forage — a place for photos that matter
Forage is a lightweight photo library built on one main principle — we need a place for images to live, not a place for them to die. Most tools are built for discovery, but become less about keeping and remembering things, and more about scrolling.

Forage is a personal visual archive meant for creative inspiration, keeping photos as memories, and nothing more. It's built for keeping things that matter.

## Priorities

- There is no algorithm — what you see is what you get
- Private by design — user images live in their own database (via a 3-minute Supabase SQL setup)
- Save from anywhere — save images either through a URL (frontend pulls from it every time) or through an upload (internal storage)

## Current State
Forage now supports mobile devices, PWA installations, and account logins as of Beta v0.25! Next — a "projects" section for creatives, a web extension, offline mode, and a standalone desktop app are all planned!
