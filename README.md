## Forage — a minimal, visual photo library
Forage is a lightweight, Pinterest/bento-inspired photo library where users can take an image, organize them by tags, and have their own "visual garden" of sorts.

## Current State
Forage now supports mobile devices, PWA installations, and account logins as of Beta v0.25! Next — a "projects" section for creatives, a web extension, offline mode, and a standalone desktop app are all planned!
