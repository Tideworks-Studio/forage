## Forage — a minimal, visual photo library
Forage is a lightweight, Pinterest/bento-inspired photo library where users can take an image, organize them by tags, and have their own "visual garden" of sorts.

## Current State
I'm currently building this first as a personal project, and it will not yet support logging in or having your own account. This repository will be meant to serve as tutorial on how to set it up on your NAS or host it on the web using a lightweight and super fast Supabase backend.
